
# src 目录解构
  - asserts —— 存放项目的img文件  asserts/[project-name]/[name].png
  - components —— 存放可公用的组件
  - directive —— 存放vue-directive 封装好的自定义指令
  - router	—— 	存放路由配置
  - store	—— 	存放状态管理器配置
  	- modules —— 各个页面的模块模板
  - styles	—— 存放公用的css样式
  - utils —— 	存放封装好的公用js 
  - views —— 存放项目视图
  - App.vue —— 项目目录
  - main.js —— 入口文件
